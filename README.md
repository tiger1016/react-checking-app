# petcheck-web

## Steps

1. git clone https://github.com/tiger1016/react-checking-app.git
2. Create .env file from .env.example
3. yarn install
4. yarn start

## How to commit

```shell
cd common
git add .
git commit -m "Description" && git push origin master
cd ..
# Can ignore above if there's no changes in common
git commit -m "Description" && git push origin master
```
